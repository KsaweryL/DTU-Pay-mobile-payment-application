# Simple DTU Pay Setup

Open workspace by running the following command in a terminal:
```bash
code "02267-Dev-of-web-services-repo.code-workspace"
````

## Tasks

There are two tasks set up in VSC:

### Install and build all 3 projects

- Open Command Palette (`Ctrl+Shift+P`)
- Select `Tasks: Run Task`
- Select `Build All (Shared, Client, Rest-Server)`


### Run Rest-Server in Dev Mode
- Open Command Palette (`Ctrl+Shift+P`)
- Select `Tasks: Run Task`
- Select `Start Quarkus Dev Mode (Rest-Server)`
